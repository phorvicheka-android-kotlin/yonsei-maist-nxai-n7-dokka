MassageScreensSharedViewModel

class MassageScreensSharedViewModel @AssistedInject constructor(val context: Context, logUseRepository: LogUseRepository, val userRepository: UserRepository, machineErrorRepository: MachineErrorRepository, val machineTimeRepository: MachineTimeRepository, sessionManager: SessionManager, sysCodeRepository: SysCodeRepository, @Assisted val navController: NavController) : MainBaseViewModel, HostModeMessageHandlerInterface

Massage screens shared view model

MassageScreensSharedViewModel contains states and methods related to:

  • Android Host Mode

  • Connection states

  • Admin Screen states

  • Communication test states

  • Error - Critical Error, Normal Error

  • Data from machine Every Seconds

  • Toggle Turn On/Off of each component

  • Internal projector

  • Assistance heater

  • External heater or External projector

  • Accessory or low frequency pad

  • Preheating

  • Firmware version check

  • User information send

  • Last usage settings of each user

  • Massage Controller Active Setting Page

  • Massage Mode

  • Hotkeys (key1 - QuickMode, key2 - Start/Pause, key3 - Preheating)

  • Scan

  • Manual Mode Settings: left and right value

  • Bluetooth Module: Audio

  • Motors (For AdminPageFunctionalTestMotorsFragment)

  • Init

  • Create log use: for user_record of machine usage

  • Override methods of HostModeMessageHandlerInterface
