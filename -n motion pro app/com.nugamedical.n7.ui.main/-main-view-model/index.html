Main view model

MainViewModel.kt manages:

  • Creation of Machine / Update idN7Code of tbl_n7

  • Check valid time user: isShowDialogDeleteOrKeepUserEvent

  • Wifi Connection

  • Audio Volume

  • Brightness

  • Massage Preheating

  • Timer

  • Permissions

  • Voice Guide

  • MediaRecorderObserver (Used for STT)

  • SpeechToText

  • AWS-IOT

  • Android Host Mode

  • Drawer and Onclick Menu Handler

  • onClickReturnBackButton

  • MassageControllerFragment Data

  • Global/Shared State related to user

  • User Registration Data (TODO: Remove this User Registration Data block after remove old registration screen)

  • Language
